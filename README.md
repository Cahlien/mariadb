# mariadb
A Python script to activate/deactivate MariaDB on Fedora33.

FILENAME
<br>
The filename variable at the top of the script is the path to the file
that will be used to track the status of the database.  Just create a
string with the path to the location of your status file.

USERNAME
<br>
The username variable at the top of the script is the username for the
MariaDB server.  Just assign your username as a string to the username
variable.
